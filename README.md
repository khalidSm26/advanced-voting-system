# Advanced Voting System

Python CLI application for managing a voter registration system with input validation.

## Features

- Register voters with age, candidate choice, and unique alphanumeric Voter ID  
- Validate voter age (18-120) and candidate selection  
- Prevent duplicate Voter IDs  
- Display all registered voters in a clean, readable format  
- Search for a voter by Voter ID  
- Interactive command-line menu for easy navigation
